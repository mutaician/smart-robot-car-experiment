# My experience with Bettlebot-3-in1-Robot 

The root project started focusing on one thing, the smartcar. I had some cool ideas especially the part of adding AI to control the car.
But the more I do so the more this project becomes more of what it started. I can say now renaming the root projeect to IOT+Robotics learning would fit well though am not sure of where my experience with this gadgets will take me so the final name would be different. 

Now to bettlebot - for this I decided to go full into arduino ide mode, to handle all of the simple but tiring code stuff when it comes with using python like my last experience with qd001 smartrobot car.
I didn't go extra as I would have wished for. the only thing I went beyond the tutorial was adding some cool animations to the 8x8 dot led matrix component. the rest was from the [Tutorial](https://github.com/keyestudio/KS0543-Keyestudio-Beetlebot-3-in1-Robot/blob/master/docs/Arduino_Tutorial/Arduino_Tutorial.md) for this no ai agent attempt integration, no new ideas to attempt
